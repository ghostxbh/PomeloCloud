# PomeloCloud
柚子云控制面板

## 演示
[https://pomelo.work](https://pomelo.work)

## 安装
### CentOS
```shell script
yum install -y wget && wget -O install.sh https://pomelo.work/env/install_1.0.sh && sh install.sh
```

### Ubuntu/Debian
```shell script
wget -O install.sh https://pomelo.work/env/install_1.0.sh && sudo bash install.sh
```
